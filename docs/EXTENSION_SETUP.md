# Browser Extension Setup

The companion browser extension adds convenient inline download buttons directly to YouTube and YouTube Music pages.

## Installation Steps (Developer / Unpacked Mode)

1. Open your browser extension management settings:
   - Google Chrome: Open a new tab to `chrome://extensions`
   - Microsoft Edge: Open a new tab to `edge://extensions`
2. Turn **Developer Mode** on (usually a toggle switch in the top-right corner).
3. Click **Load unpacked** (top-left button).
4. Select the `browser-extension` folder inside the DataSync YT Downloader directory.
5. The extension is now active! Pin it to your browser toolbar for easy access.

---

## Verifying Connection

To verify the extension can talk to your local app:
1. Open the DataSync desktop app.
2. In the setup wizard (Step 5) or the settings page, look for the **Test Extension Connection** section.
3. Click the extension icon in your browser toolbar to open the popup.
4. Click **Test Extension Connection** in the app web UI.
5. If successful, you will see a green checkmark!
