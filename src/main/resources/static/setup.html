Welcome to DataSync YT Downloader

DataSync YT Downloader downloads YouTube audio locally using yt-dlp and syncs it with your music library or shared cloud folders.

Privacy Notice: This application runs entirely locally. It does not use databases, cloud auth, or external control servers. All downloads are processed directly on this computer. YouTube links are processed by yt-dlp. Google Drive sync is handled by Google Drive Desktop.

Telegram Bot (Optional)

Configure a Telegram bot to queue downloads remotely. The bot runs locally inside this app.

1. Open Telegram. 2. Search @BotFather. 3. Send /newbot and follow instructions. 4. Copy the bot token and paste it here. 5. Search @userinfobot to get your User ID.

Browser Extension Setup

The browser extension adds an inline download button to YouTube.

To install the extension: 1. Open your browser extensions settings (chrome://extensions or edge://extensions). 2. Enable "Developer Mode" (top-right). 3. Click "Load unpacked". 4. Select the browser-extension directory in this project's folder.

Open the extension popup in your browser toolbar, then click Test.
